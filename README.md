RPC 應用範例
===================
邊學邊寫邊設計  
RPC demo across languages

#### System requirement:  
-------------------------
- go 1.11.1
- python 3.6
- grpc

#### 我的系統
-----------------------  
macOS Mojave 10.14

1. **Protocol Buffers**建立RPC service和Request Response物件
1. Homebrew安裝protobuf compile
  * brew install protobuf
1. gprc framework建立rpc應用範例，  
1. 經由Protocol Buffers使用不同程式語言建立rpc client
